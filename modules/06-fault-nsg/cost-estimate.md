# Cost Estimate: Module 06 — Fault Lab: NSG

## Summary

**Estimated cost: ~$0.00/hr** (no VM deployed in this module)

The fault lab creates only an NSG and a subnet — both free resources. You diagnose the issue using `az` CLI commands only.

| Resource | Cost |
|----------|------|
| NSG `nsg-web-fault` | Free |
| Subnet `snet-web-fault` | Free |
